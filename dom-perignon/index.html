DOM Pérignon

Named after the innovator Dom Pérignon. The DOM is the live tree of every web page's elements. This extension gives you fifteen ways to mess with it: eight live overlays that paint on top of any page, and seven theme worlds that recolor and re-type everything underneath.

  • 8Overlays — live DOM animations painted on top of the page
  • 7Theme Worlds — full graphic overrides for the page underneath
  • ×Run one of each at the same time, or just one, or neither

Get the extension

dom-perignon-v1.5.15.zip · ~120 KB · MV3 · no telemetry

A self-contained Chrome extension. No servers, no analytics. Loads in unpacked mode.

↓ Download the zip
  1. Unzip anywhere stable, e.g. Documents/dom-perignon.
  2. Open Chrome, visit chrome://extensions.
  3. Toggle Developer mode on (top right).
  4. Click Load unpacked and select the unzipped folder.
  5. Pin the toolbar icon. Click it, pick an overlay and/or a theme world. Done.
New to unpacked extensions? Chrome's walkthrough is at developer.chrome.com.
