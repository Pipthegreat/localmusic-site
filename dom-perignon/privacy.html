DOM Perignon - Privacy

Last updated: 2026-05-23

DOM Perignon stores nothing on a server, sends nothing to anyone, and tracks nothing about you. The whole policy fits on this page.

What we store

Two things, both in chrome.storage.local on your machine:

  • The hijink you've selected (e.g. champagne, ants).
  • Whether the extension is currently active (on/off).

This data never leaves your browser. We have no servers, no analytics, no third-party tracking, no account.

What we read from the pages you visit

DOM Perignon needs host_permissions: <all_urls> so it can render the selected hijink on whichever page you're viewing. Different hijinks read slightly different things from the page:

  • Logo Bounce House reads the page's favicon and meta tags to find a logo to bounce.
  • Watchers scans for images, buttons, and headings to place googly eyes on.
  • Gravity Well identifies visible page elements (images, headings) so they can fall.
  • You Got This, Champagne Bubbles, Ants, DVD Bouncer, and Aquarium don't read page content at all - they just paint over it.

Nothing read from any page is transmitted anywhere. It's used in-browser only, just long enough to render the visual overlay.

External network calls

The You Got This hijink loads the Fraunces font from Google Fonts (fonts.googleapis.com) so the motivational quotes display in the intended typeface. Google may log that request per its own privacy policy. No information about you or the page is sent - only a standard request for the font file.

No other DOM Perignon hijink makes any network call.

Permission justifications

  • storage - to remember which hijink you've picked between browser sessions.
  • host_permissions: <all_urls> - because the user explicitly selects which hijink to apply, and that hijink is rendered on whichever page the user is viewing.

Changes to this policy

If this policy ever changes (it almost certainly won't, since the extension does so little), the "Last updated" date at the top of this page will change. The current version is always at customprojects.info/dom-perignon/privacy/.
