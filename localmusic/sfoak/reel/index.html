    about lock-screen mode

    This page plays one pre-built audio file containing every preview back-to-back. Because there's no JavaScript driving the playback boundary, iOS Safari's lock-screen suspension can't break it. You'll lose per-track lock-screen metadata (the OS sees one track titled "Sfoak"), and tracks are 30-second previews instead of full songs.

    For full tracks with rich metadata, use the main sfoak page on desktop, or open the playlist in the Spotify app.
