Dual Stack

Two iframes mounted side by side, one Spotify, one SoundCloud. Each row exposes SP / SC buttons — the visitor chooses the source for each track. Only one mount renders at a time; the other hides.

Thesis: Don't guess what the visitor wants. Give them the choice and remember it. Trade: Two iframes in the page mean two API connections and bigger memory footprint. The friction of "pick a source" is real on first interaction.
