Native Audio

No iframe. The page fetches a 30-second preview MP3 from the iTunes Search API (free, no key, no rate limit issues at small scale) and plays it in a standard <audio> element. Each row also exposes hand-off links to Spotify and SoundCloud for the full track.

Thesis: The page owns the audio. Spotify and SoundCloud are escape hatches, not the substrate. Trade: Previews are 30 seconds. If iTunes hasn't indexed a track, the inline play fails — but the hand-off links keep working. The visitor's mental model is correct: this site is the launchpad, not the destination.
